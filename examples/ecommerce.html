Build personalized product recommendations using collaborative filtering and purchase history graphs

How It Works

This example demonstrates collaborative filtering using graph relationships. The graph connects:

  • Users - Customers (Alice, Bob)
  • Products - Items available for purchase (Laptop, Mouse, Keyboard, Monitor)
  • Purchases - Edges showing who bought what

The query finds products that similar users have purchased. If Alice bought a Laptop and Mouse, and Bob also bought a Laptop, we can recommend Bob's other purchases (Keyboard, Monitor) to Alice.

The Query

// Recommend products based on purchases from similar users
const recommended = await db
  .nodes('User', { id: userId })
  .follow('PURCHASED')
  .follow('-PURCHASED')
  .where({ id: { not: userId } })
  .follow('PURCHASED')
  .groupBy('id')
  .orderBy('count', 'desc')
  .limit(10)
  .exec();

Advanced Recommendations

// Frequently bought together
const bundle = await db
  .nodes('Product', { id: productId })
  .follow('-PURCHASED')
  .follow('PURCHASED')
  .where({ id: { not: productId } })
  .groupBy('id')
  .orderBy('count', 'desc')
  .limit(5)
  .exec();

// Category-based recommendations
const similar = await db
  .nodes('Product', { id: productId })
  .follow('IN_CATEGORY')
  .follow('-IN_CATEGORY')
  .where({
    id: { not: productId },
    rating: { gte: 4.0 }
  })
  .orderBy('rating', 'desc')
  .exec();

// Trending in your preferences
const trending = await db
  .nodes('User', { id: userId })
  .follow('VIEWED')
  .follow('IN_CATEGORY')
  .follow('-IN_CATEGORY')
  .where({
    created_at: { gte: '2025-01-01' }
  })
  .groupBy('id')
  .exec();
