How It Works

This example shows how sales teams can navigate complex B2B relationships. The graph connects:

  • Deals - Active sales opportunities (Q1 Deal)
  • Companies - Target organizations (Acme Corp)
  • Contacts - Decision makers and influencers (CEO, CTO, CFO, VP Sales)

The query finds C-level executives at companies with active deals in negotiation stage. This helps sales reps identify the key decision makers they need to engage.

The Query

app.ts
// Find decision makers for deals in negotiation
const contacts = await db
  .nodes('Deal', { stage: 'negotiation' })
  .follow('AT_COMPANY')
  .follow('-WORKS_AT')
  .where({ role: 'C-level' })
  .exec();

// Results show who to reach out to
console.log(contacts);
// [
//   { id: 'ceo', name: 'John Smith', role: 'CEO', company: 'Acme Corp' },

Advanced CRM Queries

app.ts
// Find warm intro paths
const introPaths = await db
  .nodes('User', { id: myId })
  .follow('KNOWS')
  .follow('KNOWS')
  .where({
    company: targetCompany,
    role: { in: ['C-level', 'VP'] }
  })
  .withPath()
  .exec();

// Track deal influence network
const influencers = await db
  .nodes('Deal', { id: dealId })
  .follow('AT_COMPANY')
  .follow('-WORKS_AT')
  .withCount('REPORTS_TO')
  .orderBy('count', 'desc')
  .exec();

// Find similar deals (past wins)
const similarWins = await db
  .nodes('Deal', { id: dealId })
  .follow('IN_INDUSTRY')
  .follow('-IN_INDUSTRY')
  .where({
    stage: 'won',
    size: { similar_to: dealSize }
  })
  .exec();

// Company hierarchy mapping
const orgChart = await db
  .nodes('Contact', { company: companyId })
  .traverse('REPORTS_TO', { direction: 'up' })
  .depth(10)
  .exec();

// Account health score
const health = await db
  .nodes('Company', { id: companyId })
  .aggregate({
    totalDeals: count('HAS_DEAL'),
    activeContacts: count('HAS_CONTACT', { active: true }),
    lastTouch: max('LAST_INTERACTION.date'),
    revenue: sum('HAS_DEAL.amount')
  })
  .exec();
